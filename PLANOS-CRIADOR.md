# Planos para enviar ao criador

## Plano 1 - MVP Venda de Celulares (2-3 semanas)
- Landing + loja com foco Redmi/Xiaomi/Realme/iPhone/Samsung.
- Cadastro/login com telefone + senha + codigo de verificacao.
- Bonus inicial de 50 creditos.
- Vitrine de promocao para novos membros.
- Troca por creditos com bloqueio de saldo insuficiente.
- Painel admin oculto com:
  - listar usuarios,
  - banir/desbanir,
  - promover/remover admin,
  - aprovar pedidos,
  - ajustar creditos com master key.

## Plano 2 - Operacao Profissional (3-5 semanas)
- Avaliacoes com estrelas + foto.
- Cadastro de produtos com imagens/videos.
- Notificacoes para cliente no painel.
- Logs completos de auditoria.
- Analytics de visitantes, cadastros e vendas.
- FAQ, garantia, reembolso e contato integrados.
- Fluxo IA triagem + abertura de ticket para humano.

## Plano 3 - Escala e Parceiros (5-8 semanas)
- Area de revendedores com dashboard dedicado.
- Bonus mensal automatico de 100 creditos para parceiro.
- Metas mensais e ranking de parceiros.
- Regras avancadas de permissao admin.
- Campanhas de desconto automatizadas.
- Otimizacao SEO regional por modelo de celular.

## Custos/infra sugerida
- Backend: Node + SQLite (MVP) -> migrar para PostgreSQL em producao.
- Hospedagem inicial: VPS basica + backup diario.
- E-mail automatizado: provedor SMTP transacional.
- WhatsApp API: integracao oficial para envio de codigo.

## Checklist de aprovacao
- [ ] Identidade visual final (logo/cores/voz)
- [ ] Politica comercial (troca/garantia/reembolso)
- [ ] Regras finais de creditos e descontos
- [ ] Dominio e hospedagem
- [ ] Textos oficiais para SEO e FAQ
